## 이미지 전처리 핵심 개념

- **리사이즈 & 정규화**: OpenCV로 해상도 변경 후 `float32`로 0~1 정규화, 저장 시 0~255 스케일 백해 JPEG/PNG 인코딩.
- **블러 스코어(Laplacian Variance)**: 이미지의 2차 미분(라플라시안) 분산을 사용해 선명도 추정. 값이 낮을수록 흐림 → 품질 필터 임계값 설정에 활용.
- **포맷 선택 트레이드오프**: JPEG(손실 압축, 용량↓, 빠른 IO) vs PNG(무손실, 용량↑). 기본은 640x640 JPEG(Q=90), 필요 시 PNG 옵션.
- **메타데이터 기반 태깅**: 파일 경로나 캡처 시간으로 시간대/도로 유형을 추정(규칙 기반). 추가 센서/날씨 메타데이터가 있으면 우선 사용.
